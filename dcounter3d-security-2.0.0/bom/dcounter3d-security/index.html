dcounter3d-security

  • Last Published: 16/10/2025 14:12 GMT
  • Version: 2.0.0

Sobre

Módulo de Segurança para Autenticação e Autorização de Usuários do Sistema.

Utiliza a especificação JAAS e dependências das biblitecas do Wildfly 37.

Wildfly

A instalação e remoção no Wildfly é realizada pelos scripts:

  • security-add.cli
  • security-remove.cli

Esses arquivos são gerados pelo Maven no diretório target do projeto.
